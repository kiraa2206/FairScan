# FairScan — AI Bias Detection Platform
### Google Solution Challenge 2026

FairScan is a browser-based AI fairness tool that detects, 
explains, and fixes hidden bias in datasets used for 
automated decisions in HR, Finance, and Healthcare.

## 🔗 Live Demo
[Click here to view FairScan](https://fairscan-2026.web.app/)

## 📋 Features
- ✅ CSV upload with real bias calculation
- ✅ Column-by-column imbalance scoring
- ✅ Interactive bias dashboard with 3 chart types
- ✅ Gemini AI plain-English explanations
- ✅ "Why is this biased?" deep-dive panel
- ✅ What-If simulator (change age/gender/location)
- ✅ One-click bias fix tool
- ✅ Before vs After comparison
- ✅ PDF audit report download

## 🛠️ Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Google Gemini 2.5 Flash API
- Chart.js, jsPDF
